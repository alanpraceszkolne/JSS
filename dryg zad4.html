1) Kiedy został stworzony JavaScript? (podaj datę).
Odpowiedź: 1995.
2) Dlaczego JavaScript jest językiem skryptowym?
Odpowiedź: JavaScript jest językiem skryptowym, ponieważ nie wymaga kompilacji do kodu maszynowego, by zobaczyć rezultaty jego działania. Wystarczy przeglądarka, która obsługuje ten język.
3) Gdzie jest wykonywany JavaScript (klient czy serwer)?
Odpowiedź: Skrypt JS jest uruchamiany po stronie klienta.
4) Dlaczego JavaScript jest bezpiecznym narzędziem?
Odpowiedź: JavaScript jest bezpieczny, ponieważ ma ograniczone uprawnienia do zasobów komputera, na którym działa. Funkcje i obiekty są wykonywane w trakcie działania programu, zapewniając kontrolę.
5) Czy JavaScript jest językiem obiektowym? Jeśli tak, to dlaczego?
Odpowiedź: Tak, JavaScript jest językiem obiektowym, ponieważ używa obiektów, metod i zdarzeń, które pozwalają opisać właściwości obiektów. W języku tym zdefiniowano wiele obiektów, metod i zdarzeń.
6) Wymień zalety JavaScriptu:
Zaleta 1) Prosta składnia, co sprawia, że JavaScript jest łatwy do nauki i stosunkowo szybko można zacząć go używać.
Zaleta 2) Jako najpopularniejszy język skryptowy w Internecie, JavaScript ma mnóstwo dostępnych przykładów zastosowań, w tym witryny, które pokazują jego funkcje.
Zaleta 3) Jego popularność wynika z uniwersalności oraz z faktu, że zmniejsza obciążenie serwerów, ograniczając ilość danych przesyłanych między serwerem a klientem.
Zaleta 4) JavaScript umożliwia tworzenie nowoczesnych aplikacji internetowych, które nie wymagają ponownego ładowania strony przy każdej interakcji użytkownika.
7) Czym jest framework?
Odpowiedź: Framework to narzędzie programistyczne do tworzenia innych aplikacji w JavaScript.
8) Jakie frameworki są używane do aplikacji mobilnych?
Odpowiedź: Jednym z popularniejszych frameworków do aplikacji mobilnych jest React Native, który umożliwia tworzenie aplikacji na różne systemy operacyjne. Inne znane frameworki front-endowe to React, Angular i Vue.
9) Co to są aplikacje responsywne?
Odpowiedź: Aplikacje responsywne to takie, które automatycznie dostosowują się do różnych rozdzielczości ekranów, urządzeń i ich rozmiarów.
10) Przepisz poniższy fragment kodu i wyjaśnij, jak działa:
‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›
type - typ pola wejściowego, np. przycisk
name - nazwa, która może być użyta w atrybucie target w linkach i formularzach. Można tu także dodać target="_blank", by otworzyć stronę w nowej karcie
value - tekst, który będzie wyświetlany na przycisku
onclick - funkcja, która zostanie wywołana po kliknięciu przycisku
11) Przepisz poniższy kod i wyjaśnij, jak działa:
window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
toolbar=no - (yes/no) decyduje o widoczności paska narzędzi
directories=no - (yes/no) kontroluje, czy przyciski katalogów będą widoczne
menubar=no - (yes/no) ustawia, czy pasek menu przeglądarki jest widoczny
height=280 - wysokość nowego okna
width=160 - szerokość nowego okna
top=200 - odległość okna od górnej krawędzi ekranu
left=200 - odległość okna od lewej krawędzi ekranu
12) Opisz funkcję window.close()
Odpowiedź: Funkcja window.close() służy do zamykania otwartego okna przeglądarki.
